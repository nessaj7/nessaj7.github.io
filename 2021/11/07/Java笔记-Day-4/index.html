  1. еҢәеҲҶзӣёеҗҢеҗҚеӯ—зҡ„зұ»
  2. еҪ“зұ»еҫҲеӨҡж—¶пјҢеҸҜд»ҘеҫҲеҘҪзҡ„з®ЎзҗҶзұ»
  3. жҺ§еҲ¶и®ҝй—®иҢғеӣҙ

    еҢ…зҡ„еҹәжң¬иҜӯжі•

    package еҢ…еҗҚ; еҢ…зҡ„жң¬иҙЁ е®һйҷ…дёҠе°ұжҳҜеҲӣе»әдёҚеҗҢзҡ„ж–Үд»¶еӨ№/зӣ®еҪ•жқҘдҝқеӯҳзұ»ж–Үд»¶

й•ҝз”Ёзҡ„еҢ…

  • java.lang.* // langеҢ…жҳҜеҹәжң¬еҢ…пјҢй»ҳи®Өеј•е…ҘпјҢдёҚйңҖиҰҒеҶҚеј•е…Ҙ
  • java.util.* // utilеҢ…пјҢзі»з»ҹжҸҗдҫӣзҡ„е·Ҙе…·еҢ…пјҢе·Ҙе…·зұ»пјҢдҫӢеҰӮдҪҝз”ЁScannerж—¶йңҖеј•е…Ҙ
  • java.net.* // зҪ‘з»ңеҢ…пјҢзҪ‘з»ңејҖеҸ‘
  • java.awt.* // жҳҜеҒҡjavaзҡ„з•ҢйқўејҖеҸ‘пјҢGUI

и®ҝй—®дҝ®йҘ°з¬Ұ

и®ҝй—®зә§еҲ« и®ҝй—®жҺ§еҲ¶дҝ®йҘ°з¬Ұ еҗҢдёҖзұ» еҗҢдёҖеҢ… еӯҗзұ» дёҚеҗҢеҢ…
е…¬ејҖ public вҲҡ вҲҡ вҲҡ вҲҡ
еҸ—дҝқжҠӨ protected вҲҡ вҲҡ вҲҡ Г—
й»ҳи®Ө вҲҡ вҲҡ Г— Г—
з§Ғжңү private вҲҡ Г— Г— Г—

жіЁж„ҸдәӢйЎ№

  1. дҝ®йҘ°з¬ҰеҸҜд»Ҙз”ЁжқҘдҝ®йҘ°зұ»дёӯзҡ„еұһжҖ§пјҢжҲҗе‘ҳж–№жі•д»ҘеҸҠзұ»
  2. еҸӘжңүй»ҳи®Өзҡ„е’ҢpublicжүҚиғҪдҝ®йҘ°зұ»пјҒе№¶йҒөеҫӘдёҠиҝ°и®ҝй—®жқғйҷҗзҡ„зү№зӮ№
  3. жҲҗе‘ҳж–№жі•зҡ„и®ҝй—®и§„еҲҷе’ҢеұһжҖ§е®Ңе…ЁдёҖиҮҙ

е°ҒиЈ…

е°ҒиЈ…д»Ӣз»Қ

е°ҒиЈ…(encapsulation)е°ұжҳҜжҠҠжҠҪиұЎеҮәзҡ„ж•°жҚ®гҖҗеұһжҖ§гҖ‘е’ҢеҜ№ж•°жҚ®зҡ„ж“ҚдҪңгҖҗж–№жі•гҖ‘е°ҒиЈ…еңЁдёҖиө·пјҢж•°жҚ®иў«дҝқжҠӨеңЁеҶ…йғЁпјҢзЁӢеәҸзҡ„е…¶д»–йғЁеҲҶеҸӘжңүйҖҡиҝҮиў«жҺҲжқғзҡ„ж“ҚдҪңгҖҗж–№жі•гҖ‘пјҢжүҚиғҪеҜ№ж•°жҚ®иҝӣиЎҢж“ҚдҪңгҖӮ

е°ҒиЈ…зҡ„зҗҶи§Је’ҢеҘҪеӨ„

  1. йҡҗи—Ҹе®һзҺ°з»ҶиҠӮпјҡж–№жі•пјҲиҝһжҺҘж•°жҚ®еә“пјү <вҖ” и°ғз”ЁпјҲдј е…ҘеҸӮж•°пјү
  2. еҸҜд»ҘеҜ№ж•°жҚ®иҝӣиЎҢйӘҢиҜҒпјҢдҝқиҜҒе®үе…ЁеҗҲзҗҶ

    е°ҒиЈ…зҡ„е®һзҺ°жӯҘйӘӨпјҲдёүжӯҘпјү

  3. е°ҶеұһжҖ§иҝӣиЎҢз§ҒжңүеҢ–private гҖҗдёҚиғҪзӣҙжҺҘдҝ®ж”№еұһжҖ§гҖ‘
  4. жҸҗдҫӣдёҖдёӘе…¬е…ұзҡ„setж–№жі•пјҢз”ЁдәҺеҜ№еұһжҖ§еҲӨж–ӯе№¶иөӢеҖј
  5. жҸҗдҫӣдёҖдёӘе…¬е…ұзҡ„getж–№жі•пјҢз”ЁдәҺиҺ·еҸ–еұһжҖ§зҡ„еҖј

    е°ҒиЈ…дёҺжһ„йҖ еҷЁ

    жһ„йҖ еҷЁдёӯи°ғз”Ёsetж–№жі•пјҢе®һзҺ°ж•°жҚ®йӘҢиҜҒйҖ»иҫ‘

з»§жүҝ

еҪ“еӨҡдёӘзұ»еӯҳеңЁзӣёеҗҢзҡ„еұһжҖ§пјҲеҸҳйҮҸпјүе’Ңж–№жі•ж—¶пјҢеҸҜд»Ҙд»Һиҝҷдәӣзұ»дёӯжҠҪиұЎеҮәзҲ¶зұ»пјҢеңЁзҲ¶зұ»дёӯе®ҡд№үиҝҷдәӣзӣёеҗҢзҡ„еұһжҖ§е’Ңж–№жі•пјҢжүҖжңүеӯҗзұ»дёҚйңҖиҰҒйҮҚж–°е®ҡд№үиҝҷдәӣеұһжҖ§е’Ңж–№жі•пјҢеҸӘйңҖиҰҒйҖҡиҝҮextendsжқҘеЈ°жҳҺз»§жүҝзҲ¶зұ»еҚіеҸҜгҖӮ

з»§жүҝзҡ„еҹәжң¬иҜӯжі•

  1. еӯҗзұ»иҮӘеҠЁжӢҘжңүзҲ¶зұ»е®ҡд№үзҡ„еұһжҖ§е’Ңж–№жі•
  2. зҲ¶зұ»еҸҲеҸ«и¶…зұ»гҖҒеҹәзұ»
  3. еӯҗзұ»еҸҲеҸ«жҙҫз”ҹзұ»

    з»§жүҝзҡ„дҪҝз”Ёз»ҶиҠӮ

  4. еӯҗзұ»з»§жүҝдәҶжүҖжңүзҡ„еұһжҖ§е’Ңж–№жі•пјҢдҪҶжҳҜз§ҒжңүеұһжҖ§дёҚиғҪеңЁеӯҗзұ»зӣҙжҺҘи®ҝй—®пјҢиҰҒйҖҡиҝҮе…¬е…ұзҡ„ж–№жі•еҺ»и®ҝй—®
  5. еӯҗзұ»еҝ…йЎ»и°ғз”ЁзҲ¶зұ»зҡ„жһ„йҖ еҷЁпјҢе®ҢжҲҗзҲ¶зұ»еҲқе§ӢеҢ–
  6. еҪ“еҲӣе»әеӯҗзұ»еҜ№иұЎж—¶пјҢдёҚз®ЎдҪҝз”Ёеӯҗзұ»зҡ„е“ӘдёӘжһ„йҖ еҷЁпјҢй»ҳи®Өжғ…еҶөдёӢжҖ»дјҡеҺ»и°ғз”ЁзҲ¶зұ»зҡ„ж— еҸӮжһ„йҖ еҷЁпјҢеҰӮжһңзҲ¶зұ»жІЎжңүжҸҗдҫӣж— еҸӮжһ„йҖ еҷЁпјҢеҲҷеҝ…йЎ»еңЁеӯҗзұ»зҡ„жһ„йҖ еҷЁдёӯз”ЁsuperеҺ»жҢҮе®ҡдҪҝз”ЁзҲ¶зұ»зҡ„е“ӘдёӘжһ„йҖ еҷЁе®ҢжҲҗеҜ№зҲ¶зұ»зҡ„еҲқе§ӢеҢ–е·ҘдҪңпјҢеҗҰеҲҷзј–иҜ‘дёҚдјҡйҖҡиҝҮгҖӮ
  7. еҰӮжһңеёҢжңӣжҢҮе®ҡеҺ»и°ғз”ЁзҲ¶зұ»зҡ„жҹҗдёӘжһ„йҖ еҷЁпјҢеҲҷжҳҫејҸзҡ„и°ғз”ЁдёҖдёӢпјҡsuper(еҸӮж•°еҲ—иЎЁ);
  8. superеңЁдҪҝз”Ёж—¶пјҢеҝ…йЎ»ж”ҫеңЁжһ„йҖ еҷЁз¬¬дёҖиЎҢ
  9. super()е’Ңthis()йғҪеҸӘиғҪж”ҫеңЁжһ„йҖ еҷЁзҡ„з¬¬дёҖиЎҢпјҢеӣ жӯӨиҝҷдёӨдёӘж–№жі•дёҚиғҪе…ұеӯҳеңЁдёҖдёӘжһ„йҖ еҷЁдёӯ
  10. JavaжүҖжңүзұ»йғҪжҳҜObjectзұ»зҡ„еӯҗзұ»
  11. зҲ¶зұ»жһ„йҖ еҷЁзҡ„и°ғз”ЁдёҚйҷҗдәҺзӣҙжҺҘзҲ¶зұ»пјҒе°ҶдёҖзӣҙеҫҖдёҠиҝҪжәҜзӣҙеҲ°Objectзұ»
  12. еӯҗзұ»жңҖеӨҡеҸӘиғҪз»§жүҝдёҖдёӘзҲ¶зұ»пјҲжҢҮзӣҙжҺҘз»§жүҝпјүпјҢJavaдёӯжҳҜеҚ•з»§жүҝжңәеҲ¶
  13. дёҚиғҪж»Ҙз”Ёз»§жүҝпјҢеӯҗзұ»е’ҢзҲ¶зұ»д№Ӣй—ҙеҝ…йЎ»ж»Ўи¶іis-aзҡ„йҖ»иҫ‘е…ізі»

    з»§жүҝзҡ„еҶ…еӯҳеёғеұҖ

еӨҡжҖҒ Polymorphic

еӨҡжҖҒеҹәжң¬д»Ӣз»Қ

ж–№жі•жҲ–иҖ…еҜ№иұЎе…·жңүеӨҡз§ҚеҪўжҖҒгҖӮжҳҜйқўеҗ‘еҜ№иұЎзҡ„з¬¬дёүеӨ§зү№еҫҒпјҢеӨҡжҖҒжҳҜе»әз«ӢеңЁе°ҒиЈ…е’Ңз»§жүҝзҡ„еҹәзЎҖд№ӢдёҠзҡ„гҖӮ

еӨҡжҖҒзҡ„е…·дҪ“дҪ“зҺ°

  1. ж–№жі•зҡ„еӨҡжҖҒпјҡ йҮҚеҶҷе’ҢйҮҚиҪҪдҪ“зҺ°еӨҡжҖҒ
  2. еҜ№иұЎзҡ„еӨҡжҖҒпјҡ
    • дёҖдёӘеҜ№иұЎзҡ„зј–иҜ‘зұ»еһӢе’ҢиҝҗиЎҢзұ»еһӢеҸҜд»ҘдёҚдёҖиҮҙ
    • зј–иҜ‘зұ»еһӢеңЁе®ҡд№үеҜ№иұЎж—¶е°ұзЎ®е®ҡдәҶпјҢдёҚиғҪж”№еҸҳ
    • иҝҗиЎҢзұ»еһӢж—¶еҸҜд»ҘеҸҳеҢ–зҡ„
    • зј–иҜ‘зұ»еһӢзңӢе®ҡд№үж—¶=зҡ„е·Ұиҫ№пјҢиҝҗиЎҢзұ»еһӢзңӢ=зҡ„еҸіиҫ№
  3. дҪҝз”ЁеӨҡжҖҒжңәеҲ¶пјҢз»ҹдёҖз®ЎзҗҶдё»дәәе–ӮйЈҹй—®йўҳ

жіЁж„Ҹз»ҶиҠӮ

  1. еӨҡжҖҒзҡ„еүҚжҸҗжҳҜпјҡдёӨдёӘеҜ№иұЎпјҲзұ»пјүеӯҳеңЁз»§жүҝе…ізі»
  2. еӨҡжҖҒзҡ„еҗ‘дёҠиҪ¬еһӢ
    • жң¬иҙЁпјҡзҲ¶зұ»еј•з”ЁжҢҮеҗ‘еӯҗзұ»еҜ№иұЎ Object obj = new Cat();

  4. еұһжҖ§/жҲҗе‘ҳеҸҳйҮҸжІЎжңүйҮҚеҶҷд№ӢиҜҙпјҒеұһжҖ§зҡ„еҖјзңӢзј–иҜ‘зұ»еһӢ
  5. instanceOfжҜ”иҫғж“ҚдҪңз¬ҰпјҢз”ЁдәҺеҲӨж–ӯеҜ№иұЎзҡ„иҝҗиЎҢзұ»еһӢжҳҜеҗҰдёәxxзұ»еһӢжҲ–xxзұ»еһӢзҡ„еӯҗзұ»еһӢ
  6. Javaзҡ„еҠЁжҖҒз»‘е®ҡжңәеҲ¶(йҮҚиҰҒ)

    • еҪ“и°ғз”ЁеҜ№иұЎеұһжҖ§ж—¶пјҢжІЎжңүеҠЁжҖҒз»‘е®ҡжңәеҲ¶пјҢе“ӘйҮҢеЈ°жҳҺпјҢйӮЈйҮҢдҪҝз”Ё

Super

superд»ЈиЎЁзҲ¶зұ»зҡ„еј•з”ЁпјҢз”ЁдәҺи®ҝй—®зҲ¶зұ»зҡ„еұһжҖ§гҖҒж–№жі•гҖҒжһ„йҖ еҷЁ

еҹәжң¬иҜӯжі•

  1. и®ҝй—®зҲ¶зұ»зҡ„еұһжҖ§пјҢдҪҶдёҚиғҪи®ҝй—®зҲ¶зұ»зҡ„privateеұһжҖ§ super.еұһжҖ§еҗҚ;
  2. и®ҝй—®зҲ¶зұ»зҡ„ж–№жі•пјҢдёҚиғҪи®ҝй—®зҲ¶зұ»зҡ„privateж–№жі• super.ж–№жі•еҗҚ(еҸӮж•°еҲ—иЎЁ);
  3. и®ҝй—®зҲ¶зұ»зҡ„жһ„йҖ еҷЁпјҢеҸӘиғҪз”ЁеңЁжһ„йҖ еҷЁзҡ„з¬¬дёҖеҸҘпјҢеҸӘиғҪеҮәзҺ°дёҖеҸҘ! super(еҸӮж•°еҲ—иЎЁ);

    дҪҝз”Ёз»ҶиҠӮ

  4. и°ғз”ЁзҲ¶зұ»зҡ„жһ„йҖ еҷЁзҡ„еҘҪеӨ„пјҲеҲҶе·ҘжҳҺзЎ®пјҢзҲ¶зұ»еұһжҖ§з”ұзҲ¶зұ»еҲқе§ӢеҢ–пјҢеӯҗзұ»еұһжҖ§з”ұеӯҗзұ»еҲқе§ӢеҢ–пјү
  5. еҪ“еӯҗзұ»дёӯжңүе’ҢзҲ¶зұ»дёӯзҡ„жҲҗе‘ҳпјҲеұһжҖ§е’Ңж–№жі•пјүйҮҚеҗҚж—¶пјҢдёәдәҶи®ҝй—®зҲ¶зұ»зҡ„жҲҗе‘ҳпјҢеҝ…йЎ»йҖҡиҝҮsuperгҖӮеҰӮжһңжІЎжңүйҮҚеҗҚпјҢдҪҝз”ЁsuperгҖҒthisгҖҒзӣҙжҺҘи®ҝй—®жҳҜдёҖж ·зҡ„ж•ҲжһңпјҒ
  6. еҰӮжһңеӨҡдёӘеҹәзұ»пјҲдёҠзә§зұ»пјүдёӯйғҪз”ЁеҗҢеҗҚзҡ„жҲҗе‘ҳпјҢдҪҝз”Ёsuperи®ҝй—®йҒөеҫӘе°ұиҝ‘еҺҹеҲҷгҖӮ

    superе’Ңthisзҡ„жҜ”иҫғ

ж–№жі•йҮҚеҶҷ/иҰҶзӣ– Override

ж–№жі•йҮҚеҶҷе°ұжҳҜеӯҗзұ»жңүдёҖдёӘж–№жі•пјҢе’ҢзҲ¶зұ»зҡ„жҹҗдёӘж–№жі•зҡ„еҗҚеӯ—гҖҒиҝ”еӣһзұ»еһӢгҖҒеҸӮж•°дёҖж ·пјҢйӮЈд№ҲжҲ‘д»¬е°ұиҜҙеӯҗзұ»зҡ„иҝҷдёӘж–№жі•иҰҶзӣ–дәҶзҲ¶зұ»зҡ„ж–№жі•

дҪҝз”Ёз»ҶиҠӮ

  1. еӯҗзұ»зҡ„ж–№жі•зҡ„еҪўеҸӮеҲ—иЎЁпјҢж–№жі•еҗҚпјҢиҰҒе’ҢзҲ¶зұ»ж–№жі•зҡ„еҪўеҸӮеҲ—иЎЁпјҢж–№жі•еҗҚе®Ңе…ЁдёҖиҮҙгҖӮ
  2. еӯҗзұ»ж–№жі•зҡ„иҝ”еӣһзұ»еһӢе’ҢзҲ¶зұ»ж–№жі•зҡ„иҝ”еӣһзұ»еһӢдёҖж ·пјҢжҲ–иҖ…жҳҜзҲ¶зұ»иҝ”еӣһзұ»еһӢзҡ„еӯҗзұ»пјӣжҜ”еҰӮ зҲ¶зұ»иҝ”еӣһзұ»еһӢжҳҜObjectпјҢеӯҗзұ»ж–№жі•иҝ”еӣһзұ»еһӢжҳҜStringгҖӮ
  3. еӯҗзұ»ж–№жі•дёҚиғҪзј©е°ҸзҲ¶зұ»ж–№жі•зҡ„и®ҝй—®жқғйҷҗгҖӮ void A() {} public void A() {}

  • ==жҳҜдёҖдёӘжҜ”иҫғиҝҗз®—з¬ҰпјҢж—ўеҸҜд»ҘеҲӨж–ӯеҹәжң¬зұ»еһӢ(еҖј)пјҢеҸҲеҸҜд»ҘеҲӨж–ӯеј•з”Ёзұ»еһӢ(ең°еқҖ)

  3. дёӨдёӘеј•з”ЁпјҢеҰӮжһңжҢҮеҗ‘зҡ„жҳҜдёҚеҗҢзҡ„еҜ№иұЎпјҢеҲҷе“ҲеёҢеҖјжҳҜдёҚдёҖж ·зҡ„

  2. йҮҚеҶҷtoStringж–№жі•пјҢжү“еҚ°еҜ№иұЎжҲ–жӢјжҺҘеҜ№иұЎж—¶пјҢйғҪдјҡиҮӘеҠЁи°ғз”ЁиҜҘеҜ№иұЎзҡ„toStringеҪўејҸгҖӮ
