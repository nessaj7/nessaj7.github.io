Java笔记——前置知识

javac 将java文件编译成class文件 Java8、Java11 （LTS)

Java特点

  1. 面向对象的OOP
  2. 健壮的：强类型机制、异常处理、垃圾自动回收
  3. 跨平台(JVM)
  4. 解释型（编译后还需要解释器来执行）

JDK

Java Development Kit，JDK = JRE + Java的开发工具。JRE（Java Runtime Environment）中包含JVM和Java的核心类库。

Java快速入门

  1. 需求说明
  2. 开发步骤
  3. 运行原理
  4. 细节和注意事项

控制台编码格式GBK

运行java hello 而非 java hello.class

注意事项

  1. 一个源文件中最多只能有一个public类，其他类的个数不限。编译后每个类都生成一个.class文件；
  2. 如果源文件包含一个public类，则文件名必须按该类名命名！
  3. 可以将main方法写在非public类中，然后指定运行非public类；

Java学习方法

  1. 先看需求
  2. 看是否能用传统技术解决
  3. 引入新技术和知识点
  4. 学习新技术或者知识点的基本原理和语法（不要考虑细节）
  5. 快速入门（基本程序，CRUD）
  6. 开始研究技术的注意事项，细节，规范，以及如何优化

基础语法

Java转义字符

\t：制表符， \n：换行符， \r：回车(光标回到行首)

注释（comment）

单行注释//、多行注释/* ... */（不要嵌套）、文档注释javadoc标签 /** ... */

Java代码规范

  1. 类、方法的注释，以javadoc的方式来写；
  2. 源文件使用utf-8编码
  3. 行宽不要超过80字符
  4. 运算符两边习惯性加空格
  5. 代码编写次行风格和行尾风格

快捷键

  1. CTRL + [ 或] 整行左右移动

DOS命令

Disk Operating System 磁盘操作系统 常见的dos命令：

  1. 查看当前目录有哪些文件 dir
  2. 切换到其他目录 cd
  3. 创建目录 md
  4. 删除目录 rd
  5. 相对路径&绝对路径 .. \
  6. 查看指定目录下所有子目录 tree
  7. 清屏clean screen cls
  8. 退出DOS exit
  9. * 其他 echo copy move del type

Java数据类型

  1. 基本数据类型 数值型 byte[1], short[2], int[4], long[8], float[4], double[8] 、字符型 char[2] 、布尔型 boolean[1]

  2. 引用数据类型 类 class 、接口 interface 、数组 []

    【String是类，属于引用数据类型】

Java API 文档

Application Programming Interface，Java提供大量的基础类。 Java类的组织形式

运算符优先级

  1. 只有单目运算符、赋值运算符是从右向左运算的。
    1. () . {} ,
    2. 单目运算符-- ++
    3. 算术运算符

    5. 比较运算符

键盘输入语句

原码、反码、补码

  1. 二进制的最高位是符号位：0表示正数，1表示负数
  2. 正数三码合一
  3. 负数的反码 = 它的原码符号位不变，其他位取反
  4. 负数的补码 = 它的反码 + 1， 负数的反码 = 它的补码 - 1
  5. 0的反码、补码都是0
  6. Java没有无符号数
  7. 计算机以补码方式运算
  8. 运算结果要看原码

数组赋值机制

数组在默认情况下是引用传递，赋的值是地址。而基本数据类型赋值方式为拷贝。
